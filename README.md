# Xueyi Wang's Personal Website

This is the source code for my personal academic website, hosted at [xueyi-wang.github.io](https://xueyi-wang.github.io/).

## About

I am a Postdoctoral Researcher at the University of Groningen, The Netherlands. This website showcases my research publications, talks, teaching activities, and project portfolio.

## Website Structure

- **Talks** -- Presentations and invited talks
- **Publications** -- Journal articles and conference papers
- **Teaching** -- Courses and teaching experience
- **Projects** -- Research projects including:
  - Large-scale Health Monitoring and User Behavior Analysis
  - Interpretable and Personalized Health Prediction System
  - Edge-Based Serious Game for Home Rehabilitation
  - Multimodal Epileptic Seizure Detection System (SeizeIT2)
  - Fall Detection Research By Egocentric Cameras
  - Sports Injury Prediction and Prevention for Professional Athletes

## Built With

This website is built using [Jekyll](https://jekyllrb.com/) and based on the [Academic Pages](https://github.com/academicpages/academicpages.github.io) template.
